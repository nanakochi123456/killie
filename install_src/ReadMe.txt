■ ブラウザー強制終了ソフト 1.1 by ななこっち★ ■
==================================================

パソコン初心者の方が、操作方法がわからなくても、
安全にブラウザーを強制終了したり、上級者がウィンドウ
の立ち上げ過ぎでブラウザーを一斉消去したり、これは
ちょっとアレな使い方であるが、親に見つかったら即
ブラウザーを消去できるようにという感じで使えるように
作成致しました。

■ 概要
Windows 上の、Internet Explorer (32bit/64bit両対応）、
FireFox、Google Chrome、Opera、Lunascape、Sleipnir、
及び正式未対応であるが、Safari を２回のアクションで
全て強制終了できるようにしたものです。

■ インストール
デスクトップにアイコンができますので、任意の位置に
アイコンを移動して下さい。

■ 使用方法
実行して、ウィンドウを確認するのみです。

■ アンインストール
スタートメニュー→コントロールパネル→プログラムと・・・
のあたりのメニューから、ブラウザー強制終了を選択すれば、
アンインストールができます。

レジストリ等は何もいじっていません。

■ 注意点
ウィンドウタイトル、及び、内部ウィンドウクラスを比較して
いますが、意図的に同じように見えるソフトは、（確認jありで）
一緒に強制終了してしまいます。

■ 動作チェック環境
Windows XP 32bit / Windows 7 64bit上で行なっています。
